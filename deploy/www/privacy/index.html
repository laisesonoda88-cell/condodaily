Política de Privacidade

Última atualização: 1 de março de 2026

1. Informações que Coletamos

O CondoDaily coleta as seguintes informações para fornecer nossos serviços:

  • Dados de cadastro: nome, e-mail, telefone, CPF/CNPJ
  • Localização: usada para check-in/check-out de serviços e busca de profissionais próximos
  • Documentos: convenções e regimentos de condomínios, analisados por IA para recomendações de serviços
  • Dados financeiros: processados exclusivamente pelo Mercado Pago — não armazenamos dados de cartão

2. Como Usamos suas Informações

  • Conectar síndicos/administradoras a profissionais de manutenção
  • Processar pagamentos e aplicar regras de multa por cancelamento tardio
  • Analisar documentos condominiais com IA para recomendar serviços
  • Enviar notificações sobre agendamentos e pagamentos

3. Compartilhamento de Dados

Não vendemos seus dados. Compartilhamos apenas com:

  • Mercado Pago: processamento de pagamentos
  • Anthropic (Claude): análise de documentos (apenas texto do documento, sem dados pessoais)
  • Profissionais contratados: dados do condomínio necessários para o serviço

4. Segurança

Utilizamos criptografia AES-256-GCM para dados sensíveis, JWT para autenticação, HTTPS em todas as comunicações e rate limiting para proteção contra abusos.

5. Seus Direitos (LGPD)

Conforme a Lei Geral de Proteção de Dados (Lei 13.709/2018), você tem direito a:

  • Acessar seus dados pessoais
  • Corrigir dados incompletos ou desatualizados
  • Solicitar exclusão dos seus dados
  • Revogar consentimento a qualquer momento
  • Portabilidade dos dados

6. Retenção de Dados

Mantemos seus dados enquanto sua conta estiver ativa. Após desativação, os dados são retidos por 5 anos conforme obrigações legais e fiscais brasileiras.

7. Contato

Para questões sobre privacidade: privacidade@condodaily.com.br
